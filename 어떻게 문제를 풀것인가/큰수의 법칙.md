### 3. 실행
#### 풀이 계획을 실행하고, 각 단계가 올바른지 점검하라.
```javascript
// 인자가 밑의 값이라고 가정
`5 8 3 
2 4 5 4 6`
```

1. 문자열 줄바꿈 기준으로 배열로 나누기 => ['5 8 3'] , ['2 4 5 4 6']
2. 띄어쓰기제거 => ['583'] , ['24546']
3. 공백을 기준으로 배열나누기 => ['5','8','3'], ['2','4','5','4','6'] 
4. 2번째 인자로 들어가는 배열의 경우 내림차순으로 정렬 => ['6','5','4','4','2']
5. number 타입으로 변환 => [6,5,4,4,2]

// 덧셈 반복 조건문

    } else {
